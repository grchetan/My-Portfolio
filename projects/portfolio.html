Overview

A fully custom-built portfolio website showcasing my skills, projects, and certifications. Designed from scratch with modern aesthetics, smooth animations, and a responsive layout that works flawlessly across all devices.

The Challenge

Building a portfolio that not only looks visually impressive but also loads fast and performs well on mobile devices. Achieving a premium feel without using any external frameworks was the core design constraint.

The Solution

I built the entire project using pure HTML5, CSS3, and vanilla JavaScript. Custom cursor animations, 3D tilt card effects, scroll reveals, and a live clock were implemented from scratch. The result is a lightweight, fast-loading portfolio with a distinctive identity.

Key Features

Custom rainbow cursor trail animation
3D perspective tilt on project cards
Dark/Light theme toggle with localStorage
Scroll-triggered reveal animations
Contact form integrated with Google Sheets
Fully responsive on all screen sizes
